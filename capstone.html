Transfer Guide Generator + Editor

I worked in a 4-person team to develop a transfer-guide generator which solves the issues of advisors having inconsitent styling and info between their transfer guides. Before our project, they would have personal word files for each guide, which led to these issues. But, with our tool there is a single template guide which they enter info into to get a completed, consitent guide for any combination of transfer major, college, and transfer college. Guide information can be saved on the server, and the lastest versions can be exported as PDF documents by the user without touching the information.
Our project uses:
  • Node.js
  • NPM
  • Vue.js 3
  • LaTeX
  • Oregon State University's Single Sign On (SAML)

Unforunately, to access the website you need to be an OSU advisor (since the system serves them exclusively) so a live preview cannot be given. 
#Setup
npm install
#Preview build (No backend)
npm run serve
#Release build
npm run build
#Host the website
node api/server.js
